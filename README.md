## Fitur Utama
- **Role-Based Access**: Pembedaan hak akses antara pengelola (Admin) dan pengguna (User).
- **Interactive Progress Graph**: Visualisasi beban latihan menggunakan Chart.js.
- **Workout Management**: Tambah, lihat, cari, dan hapus data latihan.
- **Responsive Design**: Optimal digunakan di perangkat mobile saat di gym.

##  Tech Stack
- **Frontend**: HTML5, CSS3 (Internal Glassmorphism Style), Bootstrap 5.
- **Backend**: PHP OOP.
- **Database**: MySQL.
- **Charts**: Chart.js.
